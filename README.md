# Amna-Kauser-TASK---1-Data-Science-and-Analytics-
Data Science and Analytics Internship Tasks
### Objective
Explore and visualize the Iris dataset to understand 
relationships between features and species.

### Approach
- Loaded the Iris dataset using Seaborn
- Explored data structure using shape, columns, and head
- Created 3 visualizations:
  - Scatter plot — Petal Length vs Petal Width
  - Histogram — Petal Length Distribution
  - Box plot — Petal Length by Species

### Results and Insights
- Setosa has the smallest petals (1-2 cm) and is completely 
  separate from other two species
- As petal length increases, petal width also increases
- Virginica has the longest petals overall
- Versicolor and Virginica overlap slightly in petal size
